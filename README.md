### Project Description
[CICA is a campus health inventory management system designed to monitor stock levels of sanitary pads across multiple locations. The system enables real-time stock reporting, notification logging, and centralized admin-controlled restocking.
]

### The Problem statement
[Campus health facilities struggle to track stock levels efficiently, leading to delayed restocking and lack of centralized oversight.Also students find it hard to locate them at the time of need. ]

### The Solution
[CICA lets us locate the availability of the sanitary pads and provides a stock reporting system with notification logging and an admin dashboard that enables centralized monitoring and restocking management]
